About

A regular expression is used in programming to match words, characters or patterns in strings of text. They can look pretty complex and confusing but once you know how to use them, can be extremely powerful.

This tool takes the words you would use to describe a pattern and turns it into a regular expression you can use in your code.

Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas. Vestibulum tortor quam, feugiat vitae.

Contribute

To make this little tool as awesome and as useful as possible, we need all the help we can get - so that means yours!

If you are a Reg-expert then hop onover to Github to check out all the stuff that makes this work behind the scenes. All you need to do is fork the project, hammer out your ‘translations’ and submit a pull request.
