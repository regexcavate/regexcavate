About

If you're involved in writing code of any sort, you've no doubt heard of regular expressions. But most people would agree that when you first look at them, it can feel a little like deciphering hieroglyphics.

That's why we created Regexcavate - a learning tool for digging into regular expressions and finding out how they can be used. Even the *name* of the tool is a regular expression. How meta.

The best way to learn is by doing, so have a play around. If you're a bit stuck, or feeling confused, click the tab above for a few more hints and some pre-defined patterns that you might find useful.

This tool is still in it's infancy and dealing with a pretty vast topic; if you're a regex guru then you might find it can't do some of the more complex things you know regex is capable of. If you'd like to help us make this tool better, please read about how to contribue below.

To find out more about this tool and dig deeper, check out the repo over at github.

Contribute

To make this little tool as awesome and as useful as possible, we need all the help we can get - so that means yours! We've got lots of new features in the pipeline but would really appreciate input from the community.

Add translations

If you're a Reg-expert then hop on over to Github to check out all the stuff that makes this work behind the scenes. If you'd like to add a 'translation' all you need to do is fork the project, hammer out some regex and submit a pull request.

Bugs

Bugs are everywhere; we've tried to squash as many as possible but if you find something broken, either shout at us @regexcavate on twitter or submit a fix (or both).

Improve the words

We're developers and not so much with the words. If you can explain this better (likely) or you've found a typo or non-existal word, you can fork us and submit a pull request for things like that too. Documentation is a huge deal so please feel free to contribute to that too.

Feature Requests

We have some ideas for additional features like conditionals, alternates, and translations in multiple (spoken) languages. If you have any more suggestions or requests, please let us know.
